# 🚀 YouTube 网红曝光合作系统 - 使用说明

## ⚠️ 重要提示：解决服务自动挂掉的问题

### 问题原因
之前直接运行 `python web_app.py` 会导致:
- ❌ 没有错误恢复机制，崩溃后需要手动重启

- ❌ Windows 环境下容易因网络波动、资源不足等原因退出

### ✅ 解决方案

**现在请使用以下方式启动服务:**

#### 方法 1: 双击批处理文件 (推荐)
```
双击：启动服务.bat
```

#### 方法 2: 使用 Python 脚本
```bash
python start_server.py
```

## 🔍 常见问题排查

### Q1: 服务一直重启怎么办？
**查看日志定位问题:**
```bash
# 打开最新的日志文件
logs\web_server_20260327.log
```

**常见原因:**
- 端口被占用 (5000)
- 环境变量配置错误
- 依赖包缺失

**解决方法:**
```bash
# 检查端口占用
netstat -ano | findstr :5000

# 杀死占用端口的进程
taskkill /F /PID <进程 ID>
```

### Q2: 如何彻底关闭自动重启？
编辑 `start_server.py`:
```python
MAX_RESTARTS = 0  # 设置为 0 禁用自动重启
```

### Q3: 服务挂了但没重启？
检查是否达到最大重启次数:
```bash
# 查看日志中的重启计数
grep "第 X 次尝试" logs\web_server_*.log
```

如果达到 5 次，需要手动排查错误后重新启动。
