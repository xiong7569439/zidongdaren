## 14) 邮件工具使用参考

### 14.1 系统邮件功能概览

Web 管理系统（http://localhost:5600/）提供完整的邮件工作流支持：

| 功能模块 | 说明 | 对应话术章节 |
|---------|------|-------------|
| **发送首封合作邮件** | 根据模板生成并发送首次触达邮件 | 第1、2节 |
| **邮件跟进序列** | 自动化的3天/7天/14天跟进流程 | 第4、5节 |
| **手动发送跟进** | 针对特定阶段手动发送跟进邮件 | 第4-8节 |
| **回复处理** | AI分析对方回复并生成应对话术 | 第6-8节 |
| **联系记录** | 记录所有邮件往来和联系尝试 | - |

### 14.2 邮件配置要求

**发件邮箱默认**: `cooperate@topuplive.com`

**SMTP配置（.env文件）**:
```
EMAIL_SMTP_HOST=smtp.qiye.aliyun.com
EMAIL_SMTP_PORT=465
EMAIL_SENDER=cooperate@topuplive.com
EMAIL_PASSWORD=your_password
```

**支持的邮件服务商**:
- 阿里云企业邮箱（推荐，端口465 SSL）
- Gmail（端口587 TLS）
- Outlook/Office 365
- 其他SMTP兼容服务

### 14.3 邮件发送流程

**步骤1: 查找联系方式**
- 系统自动从YouTube频道About页面提取邮箱
- 支持验证邮箱有效性（格式检查、域名验证）
- 备用联系方式：Twitter DM、Discord、Linktree

**步骤2: 选择邮件模板**
- `standard`: 标准首封触达（第1节完整版）
- `short`: 简短商务风格（第2节）

**步骤3: 填写变量**
- 游戏名称 (`{GAME_NAME}`)
- 报价锚点 (`{ANCHOR_PRICE}`)
- 引用视频 (`{VIDEO_REFERENCE_1}`)

**步骤4: 发送与记录**
- 系统自动记录发送时间、邮件内容
- 状态更新为"已发送"
- 启动跟进序列（可选）

### 14.4 跟进序列说明

系统内置自动跟进流程：

| 步骤 | 等待时间 | 触发条件 | 对应话术 |
|-----|---------|---------|---------|
| Step 1 | 首封发送后3天 | 未收到回复 | 第4节（温和跟进） |
| Step 2 | Step 1后4天 | 仍未回复 | 第5节（最后确认） |
| Step 3 | Step 2后7天 | 仍未回复 | 第12节（礼貌留存） |

**停止条件**:
- 收到对方回复（自动停止）
- 手动停止序列
- 达到最大跟进次数

### 14.5 回复处理工作流

当收到创作者回复时：

1. **在系统录入回复内容**
2. **AI自动分析回复类型**:
   - `interested`: 感兴趣，询问详情 → 使用第6节
   - `price_inquiry`: 询问价格/压价 → 使用第7节
   - `performance_guarantee`: 要求保证播放量 → 使用第8节
   - `not_interested`: 拒绝合作 → 使用第12节
   - `need_time`: 需要时间考虑 → 延后跟进

3. **系统生成建议回复**（基于对应话术模板）
4. **人工审核后发送**

### 14.6 邮件状态追踪

系统记录以下状态：
- `sent`: 已发送
- `opened`: 已打开（如支持追踪）
- `replied`: 已回复
- `failed`: 发送失败
- `bounced`: 退信

### 14.7 最佳实践建议

1. **首封邮件**: 使用简短模板（第2节），提高打开率
2. **跟进时机**: 工作日上午9-11点（美国时间）发送效果最佳
3. **个性化**: 务必引用创作者的具体视频内容
4. **报价策略**: 首封给出锚点价，预留20%议价空间
5. **停止跟进**: 3次跟进无回复后标记为"未成交"

### 14.8 常见问题

**Q: 邮件发送失败怎么办？**
- 检查SMTP配置和密码
- 确认邮箱未开启二次验证（或使用应用专用密码）
- 阿里云邮箱需使用SSL端口465

**Q: 如何批量发送邮件？**
- 当前系统支持逐个发送（避免被标记为垃圾邮件）
- 每个邮件间隔建议>5分钟

**Q: 邮件进入垃圾箱怎么办？**
- 确保SPF、DKIM记录已配置
- 避免使用过多营销词汇
- 保持邮件内容简洁专业
